# Космос.Rus

Open index.html in your browser

Статический MVP-сайт об отечественной космонавтике.

## Структура
- `index.html` — вся разметка экранов (карта планет, разделы, квиз, профиль, форум)
- `style.css` — минималистичный адаптивный стиль в заданной палитре
- `script.js` — логика навигации, прогресса, рейтинга, localStorage и аналитических хуков
- `data/content-data.js` — контент разделов и вопросы квизов
- `assets/` — локальные SVG-иллюстрации и логотип
