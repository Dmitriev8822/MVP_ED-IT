# Orbit EDU MVP (HTML/CSS/JS)

Рабочий MVP интерактивного образовательного сайта на космическую тематику.

## Что реализовано

- Авторизация и регистрация с сохранением в `localStorage`.
- Главная без классического header: навигация кликабельными планетами.
- Черно-белый минималистичный космический UI.
- Разделы:
  - История (`/sections/history`) + тест (`/sections/history/quiz`)
  - Современность (`/sections/modern`) + тест (`/sections/modern/quiz`)
  - Последние новости за месяц (`/sections/news`)
  - Истории успеха (`/sections/success`)
- Визуальное "изучение" планет: светлеют при прогрессе.
- Баллы, награды, попытки тестов, события активности.
- Профиль с подразделами:
  - `/profile/history`
  - `/profile/projects`
  - `/profile/career`
  - `/profile/achievements`
- Загрузка пользовательских достижений (метаданные файла).
- Форум с лентой тем и просмотром темы (`/forum`, `/forum/[id]`).
- "Второй экран" на главной при прокрутке вниз — упрощённый форум-preview.

## Роуты

Приложение статическое, поэтому маршруты работают через `hash`-роутинг:

- `#/`
- `#/auth/login`
- `#/auth/register`
- `#/profile`
- `#/profile/history`
- `#/profile/projects`
- `#/profile/career`
- `#/profile/achievements`
- `#/sections/[slug]`
- `#/sections/[slug]/quiz`
- `#/forum`
- `#/forum/[id]`

## Быстрый старт

1. Откройте `index.html` в браузере.
2. Зарегистрируйтесь (`#/auth/register`).
3. Перейдите на главную (`#/`) и выберите планету.
4. Изучите материалы и пройдите тест.
5. Проверьте баллы и награды в профиле (`#/profile`).

## Данные и хранение

- Контент и seed-данные: `data/content-data.js`.
- Состояние пользователей, прогресса, попыток, наград и форума: `localStorage` (`orbit_edu_mvp_v2`).

## Упрощения MVP

- Аутентификация без backend (localStorage).
- Форум без серверной модерации и real-time.
- Хранится только имя файла достижения, а не содержимое.
