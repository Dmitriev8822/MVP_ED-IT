# MVP ED-IT — Space Education

Рабочий MVP интерактивного образовательного сайта на космическую тематику.

## Что реализовано
- Регистрация/авторизация на `localStorage`.
- Главный экран без классического header: кликабельные «планеты» как навигация.
- Разделы:
  - История и современность: материалы + тесты + начисление баллов.
  - Новости за последний месяц (mock) + баллы за прочтение.
  - Истории успеха.
  - Профиль с прогрессом, наградами, активностью и загрузкой файлов-достижений (метаданные).
- Второй экран на главной — упрощенный форум.
- Хранение пользовательского прогресса, попыток тестов, наград и активности в `localStorage`.

## Быстрый старт
1. Убедитесь, что установлен Python 3.
2. В корне проекта запустите:
   ```bash
   python3 -m http.server 3000
   ```
3. Откройте `http://localhost:3000`.

## Core flow (проверка)
1. Откройте сайт и зарегистрируйтесь.
2. На главной выберите планету «История» или «Современность».
3. Отметьте материалы как изученные, затем пройдите тест.
4. Получите баллы и откройте профиль.
5. В профиле проверьте награды/прогресс и загрузите файл достижения.

## Технические детали
- Stack: HTML + CSS + Vanilla JavaScript.
- Все данные хранятся локально (`localStorage`).
- Контент, новости, тесты и форум — mock/seed данные (`data/mock-data.js`).
