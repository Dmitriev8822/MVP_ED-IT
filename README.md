# Orbit EDU MVP (HTML/CSS/JS)

Рабочий MVP интерактивного образовательного сайта на космическую тематику.

## Что реализовано
- Авторизация и регистрация на `localStorage`.
- Главная без классического верхнего header: навигация по разделам через кликабельные «планеты».
- Разделы:
  - история,
  - современность,
  - последние новости,
  - истории успеха.
- По мере изучения материалов «освещается» соответствующая планета (процент изучения).
- Тесты в разделах `history` и `modern` с фиксацией попыток и начислением баллов.
- Награды за активность: изучение разделов, чтение новостей, успешные тесты.
- Профиль с прогрессом, баллами, наградами и загрузкой достижений (имя файла + метаданные).
- Упрощенный форум + просмотр отдельной темы.
- Лента активности пользователя.

## Маршруты
SPA поддерживает следующие маршруты:
- `/`
- `/auth/login`
- `/auth/register`
- `/profile`
- `/profile/history`
- `/profile/projects`
- `/profile/career`
- `/profile/achievements`
- `/sections/history`
- `/sections/modern`
- `/sections/news`
- `/sections/success`
- `/sections/history/quiz`
- `/sections/modern/quiz`
- `/forum`
- `/forum/[id]`

## Запуск локально
> Нужен простой статический сервер (для работы client-side роутинга).

### Вариант 1: Python
```bash
python3 -m http.server 8080
```
Откройте: `http://localhost:8080/`.

### Вариант 2: VSCode Live Server
Откройте проект и запустите `index.html` через Live Server.

## Core user flow
1. Пользователь открывает сайт.
2. Переходит в `/auth/register` или `/auth/login`.
3. Выбирает раздел через планету.
4. Изучает материалы и отмечает прочтение.
5. Проходит тесты в `/sections/history/quiz` и `/sections/modern/quiz`.
6. Получает баллы и награды.
7. Видит результаты в `/profile` и подстраницах профиля.

## Структура проекта
- `index.html` — шаблоны экранов и контейнер приложения.
- `style.css` — черно-белая минималистичная космическая тема.
- `script.js` — роутинг, состояние, логика прогресса/тестов/профиля/форума.
- `data/content-data.js` — seed/mock данные: контент, новости, тесты, форум, награды.
