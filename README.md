# Orbit MVP (static)

Статический MVP-сайт на космическую тематику только на HTML/CSS/JS.

## Запуск
1. Скачайте/распакуйте папку.
2. Откройте `index.html` в браузере.

Никаких сборщиков, фреймворков и серверной части не требуется.

## Что реализовано
- Вход и регистрация (localStorage)
- Главная без верхнего header
- Кликабельные планеты-навигация по разделам
- Заметный блок новостей почти сразу на экране
- Разделы: история, современность, новости, истории успеха, тест
- Личный кабинет: история, проекты, карьера, достижения
- Баллы, прогресс, награды
- Загрузка достижений (метаданные файла)
- Простой форум
- Базовая история действий пользователя

## Структура
- `index.html` — все экраны и контейнеры
- `style.css` — монохромный минималистичный UI
- `script.js` — логика приложения и пользовательский путь
- `data/content-data.js` — mock data контента, новостей, теста и форума
