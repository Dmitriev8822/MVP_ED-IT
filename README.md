# CosmoTrack MVP (Static)

Полностью статический MVP-сайт на HTML/CSS/JS (без фреймворков), с черно-белым минималистичным интерфейсом и разделением ролей через localStorage.

## Запуск
1. Откройте `index.html` локально в браузере.
2. Зарегистрируйтесь или войдите:
   - `super@cosmo.local / 1234` (основной админ)
   - `admin@cosmo.local / 1234` (админ)

## Страницы
- `index.html` — космический фон + кликабельные планеты (навигация).
- `auth.html` — вход/регистрация.
- `section.html?section=orbit|luna|mars` — разделы с чередующимися блоками.
- `article.html?id=...` — полная статья/новость.
- `profile.html` — личный кабинет, баллы, прогресс, награды, трек, активность.
- `track.html` — загрузка резюме + статусы карьерного трека.
- `forum.html` / `topic.html?id=...` — форум.
- `admin.html` — админ-панель.

## Реализовано
- Роли: `user`, `admin`, `superadmin`.
- Ограничение доступа к админке и управлению администраторами.
- Создание новостей/статей и обновление материалов разделов через админку.
- Просмотр резюме и изменение статуса трека администратором.
- Автоматическое начисление баллов за изучение блоков, открытие полной новости и прохождение теста (без кнопок «отметить»).
- Защита от бесконечного повторного фарма баллов (one-time ключи).
- Форум с темами и комментариями.
