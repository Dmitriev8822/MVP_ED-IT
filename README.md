# SPACE MVP (статический)

Минималистичный чёрно-белый MVP-сайт на чистых HTML/CSS/JavaScript.

## Что реализовано

- Вход и регистрация.
- Главная без верхнего header.
- Кликабельные планеты как навигация.
- Заметный блок новостей почти сразу на экране.
- Разделы: история, современность, новости, истории успеха, тест.
- Личный кабинет: история, проекты, карьера, достижения.
- Баллы, прогресс, награды.
- Имитация загрузки достижений.
- Простой форум.
- Базовая история действий пользователя.
- Сохранение данных через `localStorage`.

## Как запустить

1. Склонировать/скачать папку проекта.
2. Открыть файл `index.html` в браузере.

Дополнительно можно запустить локальный сервер:

```bash
python3 -m http.server 8080
```

и открыть `http://localhost:8080`.

## Структура

- `index.html` — разметка экранов и навигации.
- `style.css` — весь стиль интерфейса (только чёрно-белая палитра).
- `script.js` — логика MVP, маршрут пользователя, localStorage.
- `data/content-data.js` — mock-данные разделов, новостей, теста и наград.
